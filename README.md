# LEGO World Map (31203) — Find the Stud

A simple web tool to find where a stud on the LEGO world map is

Note the two modes:
- Stylized
- Perfect Projection

The Stylized projection is what the lego world map looks like, the perfect projection shows what this map _did_ look like before artistic adjustments were made, so you'll have to correct a few small things (Tokyo is a good example). But at the very least it should help you find places nowhere near a coastline you'd otherwise struggle to find
